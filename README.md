# simple-interest-calculator
This project is a simple interest calculator that computes interest using the formula:

Simple Interest = (Principal × Rate × Time) / 100

## Features
- Takes principal amount
- Takes rate of interest
- Takes time in years
- Outputs simple interest

## Formula
SI = (P × R × T) / 100

## Example
If P = 1000, R = 5%, T = 2 years  
SI = (1000 × 5 × 2) / 100 = 100
